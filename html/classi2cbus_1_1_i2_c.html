I2C commands for writing to a 8-bit slave device register. All of them returns standard esp_err_t codes. So it can be used with ESP_ERROR_CHECK();.

*** WRITING interface ***

Parameters
devAddr[I2C slave device register]
regAddr[Register address to write to]
bitNum[Bit position number to write to (bit 7~0)]
bitStart[Start bit number when writing a bit-sequence (MSB)]
data[Value(s) to be write to the register]
length[Number of bytes to write (should be within the data buffer size)] [writeBits() -> Number of bits after bitStart (including)]
timeout[Custom timeout for the particular call]
Returns
- ESP_OK Success
  • ESP_ERR_INVALID_ARG Parameter error
  • ESP_FAIL Sending command error, slave doesn't ACK the transfer.
  • ESP_ERR_INVALID_STATE I2C driver not installed or not in master mode.
  • ESP_ERR_TIMEOUT Operation timeout because the bus is busy.
