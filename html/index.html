Bubo Documentation

This is a project made for the subject "electronic system design, project II" at NTNU.
The project consists of two units:
One which detects bird-collisions at the blade of the windmill, based on vibration patterns detected with an accelerometer, and sends a signal via Blluetooth Low Energy
And a second, called the hub, which relays a signal recieved via BLE to our database via WIFI.

The documentation for the unit on the wing is found in main_project, and the code for the hub is found in hub_project
To understand the project, look at the documentation of the header-files and main-files in these projects first

To run this project a few things must be done:
download ESP-IDF, this can be done from here: https://docs.espressif.com/projects/esp-idf/en/latest/esp32/get-started/. (Building and flashing the project is also decribed there)

You must also edit the file at esp-idf/components/partition_table/partitions_singleapp.csv, here "factory, app" size should be 0x180000 not 1M.

Menuconfig must be set up correctly:
-menuconfig on hub_project:
menuconfig -> example connection configuration, write name and password of the wifi network, also check that connect using wifi is set here, and not connect using ethernet
menuconfig -> component config -> bluetooth -> check the box (otherwise the project wont compile)
menuconfig -> component config -> ESP32-specific -> main XTAL frequency -> select autodetect

-menuconfig on main_project:
menuconfig -> component config -> bluetooth -> check the box (otherwise the project wont compile)
menuconfig -> component config -> ESP32-specific -> main XTAL frequency -> select autodetect
