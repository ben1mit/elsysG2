void fill_my_http_buffer_digit ( int  new_measurement)

a function for adding digits recieved in the format described in my_client_send_multiple_digits in main_project to an http-buffer this does not add "new_measurement" to the buffer. Instead it treats it as a single digit in a larger number, where all the numbers are seperated with commas. When the buffer is full it sends all its content using my_http_sender_send_turbine

Warning
this does not add "new_measurement" to the buffer!
